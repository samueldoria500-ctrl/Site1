Sobre o Projeto

Entenda os objetivos e a proposta da Entrega II.

Esta entrega tem como foco aplicar estilização avançada utilizando CSS3, explorando conceitos de design system, responsividade e componentes interativos.

O projeto demonstra domínio de técnicas modernas, como o uso de variáveis CSS, Flexbox, Grid Layout e media queries para adaptação a diferentes dispositivos.

Todo o layout foi construído seguindo boas práticas de acessibilidade, legibilidade e organização modular de código.
